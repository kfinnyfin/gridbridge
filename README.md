# GridBridge Game Landing

React/Vite project for the GridBridge landing page with the embedded GridBridge Drive gameplay demo.

## Run locally

```bash
npm install
npm run dev
```

Open the URL Vite gives you, usually:

```text
http://localhost:5173
```

## Build

```bash
npm run build
```

## Notes

- No Porsche GLB is required.
- No Python server is required.
- The game opens from the hero visual/card on the landing page.
- The game uses an embedded iframe `srcDoc` and Three.js from a CDN inside the game iframe.
