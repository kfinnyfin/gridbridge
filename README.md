# GridBridge Landing Page + Charging Demo

This is a Vite + React project for the GridBridge landing page with the embedded EV charging game.

## Local test

```bash
npm install
npm run dev
```

Open the localhost URL Vite gives you, usually `http://localhost:5173`.

## Deploy on GitHub Pages

1. Upload all files in this folder to your GitHub repo.
2. Go to **Settings > Pages**.
3. Under **Build and deployment**, set **Source** to **GitHub Actions**.
4. Push/commit to the `main` branch.
5. Open the **Actions** tab and wait for the deploy workflow to finish.
6. Your site link will appear in **Settings > Pages** and in the successful workflow summary.

Do not upload old Porsche GLB files, Python server files, vendor folders, or previous zip packages. This version is designed to run as a normal React/Vite web app.
